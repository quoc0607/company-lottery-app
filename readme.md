# 🌟 Ultimate Lottery System User Guide

Welcome to the **Ultimate Lottery System**! Designed for corporate events, year-end parties, and grand lucky draws, this application provides a professional, transparent, and highly interactive experience for both organizers and attendees.

---

## 🇬🇧 ENGLISH VERSION

### 🚀 1. Key Features
*   **🎨 Stunning Themes:** Choose from 4 immersive themes: Classic Gala, Cyberpunk, Chinese Retro, and Space Dream.
*   **🌍 Multi-Language Support:** Available in English, Vietnamese, and Chinese.
*   **📊 Smart Data Import:** Easily upload Participant and Prize lists via Excel (`.xlsx`, `.xls`) or `.csv` files.
*   **⚖️ Weighted Draw System:** Assign higher win probabilities to specific individuals (e.g., based on seniority or performance).
*   **⚙️ Flexible Draw Modes:** Choose between Auto Draw (stops automatically) and Manual Draw (controlled by the host).
*   **🏆 Glory Wall:** A beautifully categorized display of winners, highlighting Top Prizes and general prizes.
*   **🎵 Immersive Effects:** Enjoy confetti animations, dynamic card flipping, and suspenseful audio effects.

### 📁 2. Data Preparation (Excel/CSV)
To get started, prepare two Excel files. The system automatically maps columns based on their headers.

#### 👥 File 1: Participant List (Employees)
| Column Header (Keywords) | Required | Description |
| :--- | :---: | :--- |
| **ID / StaffId / Mã NV** | ✅ | Unique Employee ID. |
| **Name / Họ tên** | ✅ | Participant's full name. |
| **Dept / Bộ phận** | ❌ | Department or branch name. |
| **Weight / Trọng số** | ❌ | Win probability multiplier. Default is 1. |
| **Pool / Nhóm** | ❌ | Used to group participants for specific prize pools. |

#### 🎁 File 2: Prize Inventory
| Column Header (Keywords) | Required | Description |
| :--- | :---: | :--- |
| **Name / Tên giải** | ✅ | Name of the prize (e.g., iPhone 15 Pro Max). |
| **Tier / Hạng giải** | ✅ | Prize category (e.g., Grand Prize, 1st Prize). |
| **Count / Số lượng** | ✅ | Total quantity available for this prize. |
| **Image / Hình ảnh** | ❌ | Image URL for the prize (optional). |

### 🎮 3. Step-by-Step Instructions
1.  **Initial Setup:** Click the Gear icon (⚙️) to open Settings. Set your Event Title, upload a Logo, and choose your Theme & Language.
2.  **Import Data:** In Settings, upload your Participant and Prize Excel files.
3.  **Conduct the Draw:** Select a prize on the left, adjust the number of winners, and click **START DRAW**.
4.  **Manage Winners:** Winners appear on the Glory Wall. You can revoke a prize by clicking the trash icon (🗑️) on a winner's card.

## 🇨🇳 CHINESE VERSION (中文版)

### 🚀 1. 核心功能亮点
*   **🎨 精美主题：** 提供 4 种沉浸式视觉主题：经典盛典、赛博未来、中式复古、星空梦幻。
*   **🌍 多语言支持：** 支持中文、英文和越南语。
*   **📊 智能数据导入：** 支持通过 Excel 或 CSV 文件快速导入参与者和奖品名单。
*   **⚖️ 权重抽奖系统：** 可为特定人员设置更高的中奖概率（例如：基于司龄或绩效）。
*   **⚙️ 灵活的抽奖模式：** 支持自动抽奖和手动抽奖。
*   **🏆 荣誉榜 (Glory Wall)：** 精美展示中奖者名单，智能区分高级奖项与普通奖项。
*   **🎵 视听特效：** 包含全屏撒花动画、卡片翻转动效以及激动人心的抽奖音效。

### 📁 2. 数据准备 (Excel/CSV)
系统会根据表头（第一行）自动识别数据列。

#### 👥 文件 1：参与者名单 (员工)
| 表头关键字 (建议) | 是否必填 | 说明 / 功能 |
| :--- | :---: | :--- |
| **工号 / ID** | ✅ | 员工唯一编号。 |
| **姓名 / Name** | ✅ | 参与者全名。 |
| **部门 / Dept** | ❌ | 所属部门或分支机构。 |
| **权重 / Weight** | ❌ | 中奖概率倍数。默认为 1。 |
| **奖池 / Pool** | ❌ | 用于将参与者分组。 |

#### 🎁 文件 2：奖品清单
| 表头关键字 (建议) | 是否必填 | 说明 / 功能 |
| :--- | :---: | :--- |
| **奖品名称 / Name** | ✅ | 奖品名称（例如：iPhone 15）。 |
| **奖项 / Tier** | ✅ | 奖项级别（例如：特等奖、一等奖）。 |
| **数量 / Count** | ✅ | 该奖品的总数量。 |
| **图片 / Image** | ❌ | 奖品的图片链接 (URL)（选填）。 |

### 🎮 3. 分步操作指南
1.  **初始设置：** 点击右上角齿轮图标 (⚙️) 打开设置。输入活动标题，上传 Logo，选择主题和语言。
2.  **导入数据：** 在设置面板中，分别上传参与者和奖品的 Excel 文件。
3.  **开始抽奖：** 在左侧选择奖品，调整抽取人数，点击 **开始抽奖**。
4.  **管理荣誉榜：** 中奖者会自动保存到右侧荣誉榜。如需重抽，可点击卡片上的垃圾桶 (🗑️) 图标撤销。

## 🇻🇳 VIETNAMESE VERSION (Tiếng Việt)

### 🚀 1. Tính Năng Nổi Bật
*   **🎨 Giao diện Đa Chủ đề:** Hỗ trợ 4 giao diện tuyệt đẹp: Đại hội Cổ điển, Tương lai Cyberpunk, Trung Hoa Cổ điển, Giấc mơ Không gian.
*   **🌍 Đa Ngôn ngữ:** Hỗ trợ Tiếng Việt, Tiếng Anh và Tiếng Trung.
*   **📊 Nhập dữ liệu thông minh:** Nhập danh sách nhân viên và giải thưởng dễ dàng qua file Excel hoặc CSV.
*   **⚖️ Quay số có Trọng số:** Hỗ trợ thiết lập tỷ lệ trúng thưởng cao/thấp cho từng cá nhân.
*   **⚙️ Chế độ quay linh hoạt:** Hỗ trợ quay Tự động (Auto) và Thủ công (Manual).
*   **🏆 Bảng Vàng Vinh Danh:** Hiển thị lịch sử trúng thưởng đẹp mắt, phân loại giải Đặc biệt/Nhất và các giải phụ.
*   **🎵 Hiệu ứng Âm thanh & Hình ảnh:** Hoạt ảnh pháo hoa, thẻ bài lật mở và âm thanh hồi hộp.

### 📁 2. Chuẩn Bị Dữ Liệu (File Excel/CSV)
Hệ thống nhận diện cột tự động dựa trên tiêu đề (Header).

#### 👥 File 1: Danh sách người tham gia (Nhân viên)
| Tên Cột (Gợi ý) | Bắt buộc | Ý nghĩa / Chức năng |
| :--- | :---: | :--- |
| **Mã NV / ID** | ✅ | Mã số nhân viên (duy nhất). |
| **Họ Tên / Name** | ✅ | Tên người tham gia. |
| **Bộ phận / Dept** | ❌ | Phòng ban hoặc chi nhánh. |
| **Trọng số / Weight** | ❌ | Tỷ lệ trúng. Mặc định là 1. |
| **Nhóm / Pool** | ❌ | Dùng để chia nhóm quay thưởng. |

#### 🎁 File 2: Danh sách Giải thưởng
| Tên Cột (Gợi ý) | Bắt buộc | Ý nghĩa / Chức năng |
| :--- | :---: | :--- |
| **Tên giải / Name** | ✅ | Tên giải thưởng (VD: iPhone 15). |
| **Hạng giải / Tier** | ✅ | Phân loại giải (VD: Giải Đặc Biệt, Giải Nhất). |
| **Số lượng / Count** | ✅ | Tổng số lượng của giải thưởng này. |
| **Hình ảnh / Image** | ❌ | Link ảnh (URL) của giải thưởng (nếu có). |

### 🎮 3. Hướng Dẫn Thao Tác Từng Bước
1.  **Thiết lập ban đầu:** Nhấn vào biểu tượng Bánh răng (⚙️) để mở Cài đặt. Nhập tên sự kiện, tải Logo, chọn Theme và Ngôn ngữ.
2.  **Tải dữ liệu lên:** Trong Cài đặt, tải lên file Excel danh sách nhân viên và giải thưởng.
3.  **Tiến hành Quay số:** Chọn Giải thưởng ở cột trái, chọn số lượng người trúng, và bấm **BẮT ĐẦU QUAY**.
4.  **Quản lý Bảng Vàng:** Người trúng giải sẽ hiện ở Bảng Vàng. Bạn có thể thu hồi giải bằng cách bấm nút Thùng rác (🗑️) trên thẻ người trúng.
